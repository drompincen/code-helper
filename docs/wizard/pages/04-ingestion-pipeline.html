4 Ingestion Pipeline

Background worker processes artifacts through 6 phases

1
Pick Next Pending STORED_IN_INTAKE
Worker polls for oldest unprocessed artifact. Single-threaded, no concurrent processing.
2
Extract Text PARSING
Detect file type, read content as UTF-8 text or extract from PDF via PDFBox.
3
Normalize & Store PARSING
Remove nulls, normalize line endings, collapse whitespace. Persist to artifact_text table.
4
Chunk Text CHUNKED
Split into fixed-size windows with configurable overlap. Store in artifact_chunks.
5
Generate Embeddings EMBEDDED
TF-IDF hash vectorization per chunk. Store in chunk_embeddings as DOUBLE[].
6
Mark Indexed INDEXED
Artifact is now searchable. indexed_at timestamp set.
Failure handling: If any phase throws, status becomes FAILED with the error message stored. Original intake file is preserved for debugging.
Worker design: Single-threaded @Scheduled poll. Processes one artifact at a time. Waits for schema bootstrap before first poll.
