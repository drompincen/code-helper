Design Principles

Separation of Concerns
Upload, extraction, chunking, embedding, and search are each a separate class
Async Ingestion
Upload is fast (store & return). Processing happens in a background worker.
Chunk-Based Search
Search operates on chunks, not whole files. Better relevance and previews.
Status Tracking
Every artifact has a visible lifecycle state. Failures are recorded, not silent.
