5 Text Extraction & Normalization

Convert uploaded files into clean, searchable text

Supported Formats

Source Code
.java .kt .scala .py .js .ts .go .rs .c .cpp .rb .php .swift
Markup & Config
.xml .json .yml .yaml .html .css .sql .proto .toml .ini .properties
Documents
.txt .md .csv .sh .bat .gradle .pdf
PDF extraction uses Apache PDFBox. Text is stripped from all pages.

Normalization Steps

1
Remove null characters
\0 bytes stripped from content
2
Normalize line endings
\r\n and \r converted to \n
3
Collapse blank lines
4+ consecutive newlines reduced to 3
4
Collapse whitespace
5+ spaces/tabs within lines reduced to 4
5
Trim
Trailing whitespace per line, leading/trailing for entire text
Unsupported files (e.g., .png, .exe) cause the artifact to be marked FAILED with a descriptive error message.
