--grpc.server.port=50051

Tuning Guidance

Larger chunks (2000+)
More context per result, fewer chunks total. Better for documents.
Smaller chunks (300-500)
More precise matches, more chunks. Better for code.
Higher embedding dim (512)
Less hash collision, slightly better similarity. More memory.
