16 Quick Start

Build, run, and search in 3 minutes

# Step 1: Build
$ mvn package -DskipTests
# Step 2: Start server
$ ./run-server.sh
# Step 3: Upload your code (new terminal)
$ ./run-client.sh upload-dir \
--root ./my-project \
--ext .java,.md,.yml
# Step 4: Wait a few seconds for ingestion...
# Step 5: Search!
$ ./run-client.sh find \
--phrase "error handling" \
--mode hybrid
Prerequisites
✓ Java 21+
✓ Maven 3.9+
No external databases, no API keys, no Docker
Useful Commands
./run-client.sh healthVerify server is up
./run-client.sh statsCheck ingestion progress
./run-client.sh status --id XTrack one artifact
./run-client.sh cat --id XView extracted text
Run Tests
$ mvn test
# 48 tests, all passing
