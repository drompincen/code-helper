Global options: --host (default: localhost), --port (default: 9090)
# Upload a whole repo
$ ./run-client.sh upload-dir \
--root ./my-project \
--ext .java,.xml,.md,.yml
# Search for a concept
$ ./run-client.sh find \
--phrase "approval workflow" \
--mode semantic
# Find exact annotation
$ ./run-client.sh find \
--phrase "@Transactional" \
--mode exact
Error handling: Clean messages on file-not-found, server unreachable, and invalid args. No stack traces by default.
